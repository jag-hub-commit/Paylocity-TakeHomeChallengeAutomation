**Paylocity Employee Benefits - Automation Suite**
This project contains a comprehensive test suite for the Paylocity Employee Benefits Dashboard, covering both UI Automation and API Testing. 
It is designed to validate employee creation, benefit calculations, updates, and deletions.

**Features**
UI Automation: Uses Playwright and the Page Object Model (POM) pattern.
API Testing: Validates RESTful endpoints, status codes, and JSON schemas.
Helper Methods: Utilizes helper classes for random string generation and business logic calculations.
